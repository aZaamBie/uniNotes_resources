CSC1016 Notes:

HCI:

The study of Human-Computer-Interaction. Least interesting Module in CSC1016S.


I've summarized ALL of the HCI slides. The last lecture contains some info generated by chatGPT.
This is more text-heavy. Contains few images.

UML:

Unified Modelling language. More helpful and slightly more interesting than HCI.


I've summarized the UML slides. There's quite a few images, showcasing the different relationships, assocations, diagrams etc. There's also a section at the end which contains notes from GeeksForGeeks if you're not satisfied with the lecture content.

GUI

Generated User Interface. Probably the most interesting and most practical module here. You can create small programs, apps with GUI.

Best way to learn this is tinkering with GUI in Java, looking at how the components work with one another, and how the parameters affect certain components.
